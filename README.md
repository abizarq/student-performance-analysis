# Student Performance Analysis

## Objective
Analyze student data using Pandas to clean, process, and extract insights.

## Features
- Handled missing data using fillna() and dropna()
- Created grade column using apply()
- Performed analysis using groupby(), value_counts()
- Identified top-performing students

## Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Key Insights
- Found average marks across cities
- Identified top students (Grade A)
- Cleaned dataset for accurate analysis
