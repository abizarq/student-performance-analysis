# Student Performance Analysis

## Objective
Analyze student data, perform visualization, and build a basic machine learning model to predict student marks.

## Features
- Handled missing data using fillna() and dropna()
- Created grade column using apply()
- Performed analysis using groupby(), value_counts()
- Identified top-performing students

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Insights
- Found average marks across cities
- Identified top students (Grade A)
- Cleaned dataset for accurate analysis

## Machine Learning

- Applied Label Encoding to convert categorical data
- Selected features (age, city) and target (marks)
- Trained Linear Regression model
- Predicted student marks based on input features
